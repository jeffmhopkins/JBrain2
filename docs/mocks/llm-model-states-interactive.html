On-box model states

Same card, same CSS. Tabs run Resident → Available → Catalogue (live first, broadest last). Staged is no longer a tab — it's a transient preview: tap Stage on an available model to see, on the memory bar, whether loading it evicts something.

The four states

Catalogue

Every known model — installed (weights on disk) or uninstalled.

Available

Installed models the router may swap in and out. Only these compete for memory.

Resident

Loaded in unified memory right now.

Staged

A preview, not a stored state. Staging runs the eviction check and shows what would leave to make room — commit with Load, or Cancel and nothing changed.
