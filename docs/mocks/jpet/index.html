JPet — aesthetic mockups

A Tamagotchi-style robot pet with Furby babble, in a Tron/synthwave wireframe room. One real 3D take plus five flat-sketch skins for the same toy — pick the vibe before we build anything real.

Every mockup is live and interactive: move the mouse (its eyes follow), click the robot to poke it, and use the buttons to Feed / Play / Pet / Sleep / Talk. Food·Energy·Fun·Love drain over time (Tamagotchi care-loop); neglect it and it gets hungry and sad. Hit 🔊 Sound then 💬 Talk to hear the Furby babble.

🤖
★ 3D ROOM · WEBGL
Room 3D — the real thing
A true 3D perspective room with the pet as a wireframe robot that walks around the floor, turns to face where it's going, and reacts. Click the floor to send it, click it to poke, drag to orbit. This is the direction the actual build takes (Three.js/WebGL). The five below are flat 2D sketches for palette comparison.
📱
★ PHONE CONTROL · PWA
Control screen — the kids' remote
The mobile "remote" that pairs to the Wall: live pet status + Needs meters, care buttons, a room map to send it places, a chat log, and a talk bar with a mic. Best viewed on a phone (or a narrow window).
