Task grouping · C — swimlane board
One group per screen. Each group is a full-width lane you swipe between (pager dots up top); the last panel is ＋ New lane. Drag a card to reorder within its lane, or drag it to the screen edge to carry it into the neighbouring group. Fewer tasks visible at once, but each group gets its own uncluttered space — the Kanban mental model on a phone.
